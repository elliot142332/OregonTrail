Oregon Trail Video Game - Simplified
Java 1 Final Projects

This project is a simplified version of the original Oregon Trail video Game

The project is divided into 5 different classes
-OregonTrail.java
-Player.java
-Images.java
-Store.java
-RandomEvents.java

The project has a image folder containing all the pictures seen in the game.

In order to run the game, navigate to the SRC file and then load the OregonTrail.java class
